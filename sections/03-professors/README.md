# Relationships with professors

**Format:** synthesized section with attributed quotes.

## What goes here

- How to start a real relationship with a professor (office hours, reading courses, asking for problems)
- How reading courses actually work at UT, logistically and intellectually
- UT-specific notes: which research groups are active, which faculty take undergrads seriously in which areas
- How to ask for letters of recommendation, and how to make sure they'll be strong
- What a strong letter actually requires from the writer (and therefore from you)
- Mistakes to avoid

Synthesized by the steward from contributor responses, with attributed quotes.
